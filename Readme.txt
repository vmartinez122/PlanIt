PARA EJECUTAR EL PROGRAMA:
Ejecutar Start con java desde la consola, escribiendo su dirección con "/".
Se ha de ejecutar el programa desde fuera de la carpeta PlanIt.

Ejemplo:
C:\Usuario\Documents> java PlanIt/Start

*Es posible que la versión de java instalada no sea compatible con la que se ha compilado el programa.

INSTRUCCIONES DE USO:
Al entrar al programa encontraremos el siguiente menú:
   1. New event
   2. Edit event
   3. Delete event
   4. Show events
   5. Exit

Para comunicarnos con el programa, introduciremos por consola el número que corresponda a la función .
Dentro de cada función, contestar al programa los datos que pide, si un dato tiene el formato incorrecto,
el programa pedirá que se vuelva a introducir.

El programa no accepta carácteres especiales como accentos o "ñ,ç"

Las fechas tienen un formato dd/mm/yyyy, con límites para cada tipo de dato:
   Días:     1 - 31 
   Meses:    1 - 12
   Años:  1000 - 9999

Siempre que vayamos a hacer una modificación, el programa pedirá una confirmación final:
Do you want to add: [ejemplo] (y/n)?
Para aplicar los cambios, debemos introducir una y/Y, cualquier otro carácter o un enter, anulará los cambios.
